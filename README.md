# English

## 🚀 What is Apolo?

Apolo is an open-source AI assistant and agent platform designed to be much more than a traditional chatbot.

Its goal is to provide a unified ecosystem where users can interact with artificial intelligence, automate tasks, manage information, control devices, develop software, and create custom AI agents.

Apolo is built with modularity, flexibility, and self-hosting in mind, allowing it to run on personal computers, servers, cloud environments, and future embedded devices such as Raspberry Pi systems and smart home controllers.

## 🏗 Architecture

Apolo follows a modular architecture that separates each component into independent services.

Possible modules include:

- Frontend Interface
- Backend API
- AI Engine
- Memory System
- Agent System
- Automation Engine
- Plugin Manager
- Device Integration Layer

This structure allows developers to extend Apolo without modifying the entire system.

# Português

## 🚀 O que é o Apolo?

Apolo é uma plataforma open-source de assistente e agentes de inteligência artificial criada para ir muito além de um chatbot tradicional.

Seu objetivo é oferecer um ecossistema unificado onde usuários possam interagir com IA, automatizar tarefas, gerenciar informações, controlar dispositivos, desenvolver software e criar agentes personalizados.

O Apolo é desenvolvido com foco em modularidade, flexibilidade e auto-hospedagem, permitindo execução em computadores pessoais, servidores, ambientes em nuvem e futuramente dispositivos embarcados como Raspberry Pi e controladores de automação residencial.
